# LLM Data Filter

Simple filter for datasets based on LLMs.

## Dependencies

Python Version `>= 3.10`

```bash
pip install -r requirements.txt

pip install -e path/to/flex-infer
```

## Usage

Set the model paths in the `filter_dataset.py` file.

```bash
chmod +x run_script.sh

./run_script.sh <model_name>
```

Different models:

- Llama-3.1 8B
- Mistral-v2 7B
- Gemma-2 9B
